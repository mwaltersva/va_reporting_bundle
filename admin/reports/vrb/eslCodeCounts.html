LEP Students by ESL Status Code

Counts students with an ESL services code who were active on the entered snapshot date. If no snapshot date is entered, the current date will be used. Usual snapshot dates include:

  • October 1st - Fall SRC
  • December 1st - Dec 1 Special Ed child count
  • March 31st - Spring SRC
  • Last day of School - EOY SRC
